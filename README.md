# GreenMail client integrations

This project helps integrate [GreenMail](https://github.com/greenmail-mail-test/greenmail) with various email clients.

* [Roundcube](roundcube)
* [Rainloop](rainloop)
* [SquirrelMail](squirrelmail)
* [SnappyMail](snappymail)
